下载下来后，需要如下步骤启动项目：
- 打开终端
- 输入以下命令
```
cd angular-demo

npm start
```
当然，你首先要安装`@angular/cli`，如果还没安装，可以看第一章《[Angular CLI](http://ghmagical.com/article/page/id/na7zoYaTCXmK)》。
